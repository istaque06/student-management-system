# 🎓 Student Management System (C++)

A simple and efficient console-based Student Management System built using C++.

## 🚀 Features
- Add Student Details
- Display Student Records
- Search Student by ID
- Delete Student Record

## 🛠️ Tech Stack
- C++
- Data Structures (Vector, Structures)

## 💡 Concepts Used
- Object-Oriented Programming (OOP)
- STL (Standard Template Library)
- CRUD Operations

## ▶️ How to Run
1. Open in VS Code
2. Compile:
   g++ student_system.cpp -o student
3. Run:
   ./student
